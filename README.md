# Euphoria

Full-stack e-commerce web application with product catalog, authentication, cart system, and role-based admin dashboard.

Demo: https://meek-bunny-684a27.netlify.app

## 🧰 Tech Stack

React, Vite, Node.js, Express, MongoDB, Mongoose, JWT, SCSS

---

## ✨ Features

- Product catalog with filtering, search, sorting, pagination
- Product detail page with image gallery and variants
- Guest and authenticated cart system with migration support
- JWT authentication
- Role-based admin dashboard (admin / manager)
- Product and user management
- Image upload support

---

## 🏗 Architecture

Client (React SPA) + REST API (Express + MongoDB)

---
